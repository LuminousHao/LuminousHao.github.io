前言

最近无意间刷到一些关于笔记的视频，让我这个高三以来已经多年不怎么写笔记的人一下来了兴趣，在好奇心的驱使下跟着视频内容进行操作，但中途还是遇到了一些奇奇怪怪的问题，好在最后都解决了。最后发现确实不用怎么写笔记了，以后应该是敲笔记才对吧。

记忆的过程

  • 知识被大脑记住的过程 知识被大脑记住的过程是一个复杂的生理和心理过程，主要分为三个阶段：

    • 编码阶段： 大脑将信息转换为神经信号，并将其存储在短期记忆中。
    • 巩固阶段： 短期记忆中的信息通过重复或意义编码被转移到长期记忆中。
    • 检索阶段： 当需要回忆信息时，大脑会从长期记忆中检索相关信息。
      来源

  • 💡 影响记忆的关键因素 影响记忆的关键因素包括：

    • 注意： 对信息进行充分的注意是记忆的前提。
    • 意义编码： 将信息与已有的知识联系起来，使其更有意义，更容易记忆。
    • 重复： 重复信息可以加强记忆痕迹。
    • 情感： 情感强烈的信息更容易被记住。
      来源

  • 📚 学习的终极奥秘 学习的终极奥秘在于如何将知识转化为长期记忆。一些有效的学习方法包括：

    • 检索练习： 通过回忆来巩固记忆。
    • 间隔重复： 在不同的时间间隔内重复学习信息。
    • 多样化练习： 使用不同的学习方式来加强记忆。
      来源

从上面的内容中我们可以发现重复的力量是相当强大的，其实也就是复习。那么有没有一种笔记方式，能让我们快速获取笔记内容并且能够将重要内容进行有效的重复呢？而这个问题也正是本文所探讨的核心，我们应该如何实现

自己动手丰衣足食？

从纸质笔记到电子笔记，各有优势利弊。在某些情况下，人们仍然会选择使用纸质笔记。然而，目前电子笔记的使用频率越来越高，越来越多的人开始依赖于电子笔记。但是，随着电子笔记的普及，一些问题也逐渐显现出来，例如越来越多的笔记软件采用付费模式，以及充斥着大量无用内容的现象。在商业化的推动下，一些厂商或许已经忘记了最初的用户需求。理想中的笔记软件应该具备一些基本特点，比如能够快速获取基本的笔记内容、页面简洁明了、笔记管理和使用方法简单易懂、能够实现笔记之间的关联性，以及能够帮助用户提取和复习知识重点。而目前市面上的免费的笔记软件很难同时满足以上的需求，好在，自己动手便可以丰衣足食

软件介绍

Obsidian

什么是Obsidian

Obsidian，是使用Markdown语法的闭源笔记软件。用户可以在笔记间建立双向链接，并且可以神经网络图的形式将这些链接可视化。该笔记旨在帮助用户以一种灵活的、非线性的方式来组织和构架他们的想法和知识。

Obsidian的优势

  • 双链笔记：Obsidian 的双链笔记功能使得笔记之间可以相互关联，这在传统笔记软件中并不常见。这种双向链接可以帮助用户更好地组织和理解复杂的信息关系。

  • Markdown 支持：Obsidian 支持 Markdown 语法，这使得用户可以使用简单的标记语言来编辑和格式化笔记，而不必依赖复杂的编辑界面。

  • 本地优先：Obsidian 采取本地优先的存储方式，用户的笔记数据存储在本地设备上，而不是云端。这样可以确保数据的安全性和隐私性。

  • 插件生态：Obsidian 具有丰富的插件生态系统，用户可以根据自己的需求选择安装各种插件来扩展软件的功能。

  • 自定义性：Obsidian 提供了丰富的自定义选项，用户可以根据自己的喜好调整界面、主题和其他设置。

Anki

About Anki

Anki 是一个让记忆变得容易的程序。因为它比传统的学习方法效率高得多，所以您可以大大减少学习时间，也可以大大增加学习量。
任何在日常生活中需要记住事情的人都可以从 Anki 中受益。由于它与内容无关，并且支持图像、音频、视频和科学标记（通过 LaTeX），因此可能性是无穷无尽的。

Features

  • Synchronization: 使用免费的AnkiWeb同步服务，让您的卡在多个设备之间保持同步。

  • Flexibility: 从卡片布局到复习时间，Anki 有丰富的选项供您自定义。

  • Media-Rich: 在卡片上嵌入音频剪辑、图像、视频和科学标记，并精确控制其显示方式。

  • Optimized: Anki: 将毫无问题地处理 100,000+ 张牌的套牌。

  • Fully Extensible: 有大量可用的附加组件。

  • Open Source: 由于代码和存储格式是开放的，因此您的重要数据是安全的。

Start

进入Anki官网,点击Download跳转到系统选择页面，选择对应的 Qt6 版本下载即可。安装打开之后继续注册账号并登录。

一些前置设置

在Obsidian安装插件

2.进入浏览器打开Omnivore >> 点击左下角账号 >> API Keys >> 复制当前浏览器的Key

3.返回Obsidian >> 继续打开设置 >> 在最下方 第三方插件 中找到Omnivore >> 在第一行 API Keys 中粘贴复制的Key

获取笔记

1.当浏览器成功安装Omnivore拓展之后右上角会出现一个黄色的小图标

2.在浏览器打开需要记录笔记的网站，点击Omnivore图标后会自动将网页内容存入inbox


3.在inbox找到对应文章，在阅读时对重要内容进行高亮标记和注释，插件会自动将文章中高亮和注释内容同步到Obsidian中

4.当一篇文章阅读完之后，可以将其存入 Archive 防止遗失，同时也可以为文章设置不同的标签来提高管理效率

善用高亮
